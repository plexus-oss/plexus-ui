# Contributing to Plexus UI

Thanks for your interest in Plexus UI! 👋

Plexus UI is published as a **resource and reference library** — a set of
components you can install, copy, and adapt into your own projects. It is
maintained by the core team and is **not** a collaborative open-source project.

## How you can contribute

✅ **Open an issue.** Bug reports and well-scoped feature requests are welcome
and genuinely valued:

- 🐛 [Report a bug](https://github.com/plexus-oss/plexus-ui/issues/new)
- 💡 [Request a feature or change](https://github.com/plexus-oss/plexus-ui/issues/new)

## What we don't accept

❌ **Pull requests from outside the core team.** To keep the library coherent
and easy to maintain, we don't merge external PRs. Any PR opened by a
non-collaborator is closed automatically with a pointer back here. This isn't
personal — it's how we keep the project sustainable as a small team.

If you need a change, the fastest path is an issue describing the problem or
use case. If you want to build on Plexus UI for your own needs, the MIT license
lets you fork and modify freely.
